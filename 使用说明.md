# 微信开发者工具使用说明

## 一、下载安装

1. 访问微信开发者工具官网：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 根据你的操作系统（macOS/Windows）下载对应版本
3. 安装并打开微信开发者工具

## 二、导入项目

### 方法一：直接导入
1. 打开微信开发者工具
2. 点击左侧「+」号或「导入项目」按钮
3. 填写项目信息：
   - **目录**：选择项目文件夹
     ```
     /Users/leehiukit/CodeBuddy/20260605171656
     ```
   - **AppID**：填写你的小程序 AppID
     - 如果没有，点击「测试号」使用测试号
     - 或者去 https://mp.weixin.qq.com 注册小程序获取
   - **项目名称**：二手笔记本商城
4. 点击「导入」按钮

### 方法二：命令行打开
```bash
# macOS
open -a "wechatdevtools" /Users/leehiukit/CodeBuddy/20260605171656

# 或者使用 cli（需要先配置环境变量）
cli open --project /Users/leehiukit/CodeBuddy/20260605171656
```

## 三、配置 AppID

### 获取 AppID
1. 访问 https://mp.weixin.qq.com
2. 登录你的小程序账号
3. 在「开发」->「开发管理」->「开发设置」中找到 AppID

### 使用测试号（推荐新手）
1. 在导入项目时，点击「测试号」
2. 系统会自动分配一个测试 AppID
3. 测试号可以体验大部分功能

## 四、项目预览

导入成功后：
1. 左侧模拟器会自动显示小程序界面
2. 可以在模拟器中点击操作预览效果
3. 底部控制台可以查看日志和错误信息

## 五、真机预览

1. 点击工具栏「预览」按钮
2. 生成二维码
3. 用微信扫码即可在手机上体验

## 六、常见问题

### Q: 提示 "app.json: No such file or directory"
A: 确保选择的是项目根目录，不是上层目录

### Q: 提示 "Invalid AppID"
A: 检查 AppID 是否正确，或使用测试号

### Q: 页面显示空白
A: 检查控制台是否有错误信息，通常是图片资源缺失

### Q: 如何添加图片资源？
A: 在项目目录下创建 `images` 文件夹，按以下结构放置图片：
```
images/
├── tab/          # 标签栏图标
├── icon/         # 功能图标
├── banners/      # 轮播图
└── products/     # 商品图片
```

## 七、开发调试

### 修改代码
- 直接在微信开发者工具中编辑文件
- 保存后自动编译刷新

### 查看日志
- 点击「调试器」->「Console」查看日志
- 使用 `console.log()` 输出调试信息

### 网络请求
- 点击「调试器」->「Network」查看网络请求
- 需要在小程序后台配置合法域名
